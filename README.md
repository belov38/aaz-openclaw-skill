# aaz-openclaw-skill

OpenClaw skill for AAZ AI Proxy image generation (Google nano-banana models).

## Install

```bash
mkdir -p ~/.openclaw/skills/aaz-image-gen
curl -sL https://raw.githubusercontent.com/belov38/aaz-openclaw-skill/main/SKILL.md \
  -o ~/.openclaw/skills/aaz-image-gen/SKILL.md
```

## Files

- `SKILL.md` — Skill definition (frontmatter + image generation docs)
- `update.md` — Platform updates (checked by heartbeat)
